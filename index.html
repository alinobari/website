Text Compression/Encryption

Python Web Application

Using the power of Python, this web application compresses/encrypts, and decompresses/decrypts a text file. Using the webpage, a .txt file can be submitted, resulting in a compressed/encrypted file being downloaded in a .mtf format. This file can then be submitted again into the webpage to decompress and decrypt the file back into its original .txt format. The encrypted file is given a 4 value character to check authenticity of the file during decryption.
